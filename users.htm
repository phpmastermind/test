Mokoala is a CMS with a front-end API for accessing stored data. Installation is easy as long as you have PHP 5 and MySQL available on your server.

Simply copy the contents of the /Source/ folder to the directory you want to install the script. Once you have uploaded your site use your browser to load up {upload_path}/admin/ to begin the insallation wizard.

Video of Installation

Step-by-Step Guide

Requirements

The first screen you are shown is a check-list that needs to be completed before continuing.

  1. PHP 5 or above
  2. The permissions of folder "../tpl/uploads/" set to 0777
  3. The permissions of folder "../tpl/img/thumbs/" set to 0777
  4. The permissions of folder "../admin/resources/backups/" set to 0777
  5. The permissions of folder "../admin/resources/restore/" set to 0777
  6. The permissions of file "../admin/config.ini.php" set to 0777

Step 1: Database

The next step requires you to enter your MySQL server details. Some installations will have a list of optional 'Components' that can also be installed. If this applies simply tick the boxes of the components you wish to install.

Step 2: Site Details

The next step requires you to enter details of your site. It's important that you enter a working email address for the server that you are installing Mokoala on as it will be used for sending emails.

Step 3: Your Account

These will be your administrator account login details. You will use this email and password as your Mokoala login credentials.

Step 4: Summary

The final step is a summary of each step. You can go back and change anything that you have entered, if needed.

Finished

And that's it. You are now ready to use your installation of Mokoala!
